如果代码目录结构类似于：

``` linux
- code
  - pkg1
    - __init__.py
    - module.py
  - pkg2
    - exp.ipynb
```

注意到pkg1是一个python包（有__init__.py），但是pkg2不是（__init__.py），因此相对导入（exp.ipynb导入module）失效，并且即使有__init__.py，在测试中jupyter也引用不了。

会出现错误：

`ImportError: No module named 'pkg1'`

解决方法是，在导入代码前加入代码：

``` python
import sys
sys.path.append('..')
```

这样就可以写：

``` python
import pkg1 import *
```
